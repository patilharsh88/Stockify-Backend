-Overview

  Stockify is a backend inventory management system built using Spring Boot, Hibernate (JPA), and PostgreSQL.
  It allows businesses to manage items, track stock levels, and record stock transactions efficiently.

The system supports:

    Item management (CRUD)
    
    Stock IN / OUT transactions
    
    Automatic stock updates
    
    RESTful APIs
    
    Clean layered architecture

#Tech Stack

  Technology	Purpose
  Spring Boot	Backend framework
  Hibernate / JPA	ORM & persistence
  PostgreSQL	Database
  Lombok	Boilerplate reduction
  Maven	Dependency management
  Postman	API testing
